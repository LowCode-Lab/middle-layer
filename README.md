# middle-layer

## 现阶段需要支持的基础控件

### 文本输入

+ 单行文本框(input)
+ 多行文本块(textarea)

### 选项选择

+ 单选框(radio)
+ 复选框(checkbox)
+ 开关(switch)
+ 下拉选择框(select)
+ 弹出选择框(selectbox)

### 日期时间

+ 时间选择器(time-picker)
+ 日期选择器(date-picker)
+ 年份选择器(year-picker)
+ 月份选择器(month-picker)
+ 日期时间选择器(datetime-picker)

### 附件上传

+ 文件上传(file-upload)
+ 图片上传(image-upload)

## 现阶段需要支持的复合控件

+ 时间范围选择器(time-range-picker)
+ 日期范围选择器(date-range-picker)
+ 年份范围选择器(year-range-picker)
+ 月份范围选择器(month-range-picker)
+ 日期时间范围选择器(datetime-range-picker)

## 下一阶段需要支持

+ 穿梭框(transfer)
+ 树(tree)
+ 列表(list)
+ 表格(table)
